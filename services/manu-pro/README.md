# manu-pro

Chat tipo wizard para recolectar informacion paso a paso.

## Estructura
- `src/` codigo fuente
- `src/routes/` endpoints/API
- `src/lib/` logica del wizard
- `public/` assets estaticos
- `assets/` recursos de diseño
- `config/` configuraciones
- `types/` tipos/contratos
- `tests/` pruebas
